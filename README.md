# SentinelGPT: GPT-Powered SOC Analyst Copilot

## Overview

SentinelGPT is an AI-powered Security Operations Center (SOC) analyst assistant that automates security log analysis, threat classification, MITRE ATT&CK mapping, and incident report generation.

The system processes Windows and Linux security logs, detects suspicious activity, and leverages a local Large Language Model (LLM) through Ollama to generate analyst-ready incident reports.

## Features

* Security log ingestion and parsing
* Brute-force attack detection
* Privilege escalation detection
* Suspicious PowerShell activity detection
* Unauthorized account creation detection
* MITRE ATT&CK technique mapping
* AI-generated incident summaries
* Containment and remediation recommendations
* Downloadable incident reports

## Technologies

* Python
* Streamlit
* Ollama
* Llama 3
* MITRE ATT&CK Framework
* Pandas

## Sample Workflow

1. Upload security logs
2. Detect suspicious activity
3. Map events to MITRE ATT&CK techniques
4. Generate AI-powered incident reports
5. Export investigation findings
